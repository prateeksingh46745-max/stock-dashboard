# 📈 Real-Time Stock Market Dashboard

A powerful, interactive stock market dashboard built with Python and Streamlit.
Track live stock data, technical indicators, and market news across global markets.

## 🎯 Project Highlights

- Multi-market stock tracking (India, US, UK, Japan)
- Real-time stock data using Yahoo Finance
- Interactive Plotly visualizations
- Technical analysis with RSI, Moving Averages, and Bollinger Bands
- Stock comparison functionality
- News integration
- CSV export support

## 🌍 Supported Markets
- 🇮🇳 India (NSE/BSE) — NIFTY 50, SENSEX, Reliance, TCS, Infosys and more
- 🇺🇸 US (NYSE/NASDAQ) — S&P 500, Apple, Tesla, NVIDIA and more
- 🇬🇧 UK (LSE) — FTSE 100, HSBC, BP, AstraZeneca and more
- 🇯🇵 Japan (TSE) — Nikkei 225, Toyota, Sony, Nintendo and more

## ✨ Features
- 📊 Live candlestick charts with volume
- 📉 Technical indicators — MA20, MA50, RSI, Bollinger Bands
- 💱 Auto currency switching — ₹ £ ¥ $
- 📰 Latest news feed per stock
- 📊 Stock comparison tool
- 📥 CSV data export
- 🌙 Clean dark theme UI

## 🛠️ Tech Stack
| Tool | Purpose |
|------|---------|
| Python | Core language |

## 🚀 Run Locally

**1. Clone the repo**
```bash
git clone https://github.com/Prateeksinghchouhan12/stock-dashboard.git
cd stock-dashboard
```

**2. Create virtual environment**
```bash
python -m venv venv
venv\Scripts\activate
```

**3. Install dependencies**
```bash
pip install -r requirements.txt
```

**4. Run the app**
```bash
streamlit run app.py
```
